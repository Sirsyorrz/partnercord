# Partnercord вЂ” Sea of Thieves Creators Discord Archive

An interactive static web app that presents the full Sea of Thieves Creators Discord server archive (2019вЂ“2026) as a browsable, searchable Discord-like interface. Deployable to GitHub Pages with zero runtime dependencies.

## Features

- Discord-faithful UI with 9 logical channels spanning both export periods
- Merged duplicate channels (same channel, renamed across exports)
- Full-text and username search across all channels
- Statistics home page: message leaderboards, channel activity, keyword tracker, reaction totals, timeline
- Pagination (200 messages per page) handles the massive `general` channel

## Running Locally

**Parse the Discord exports into JSON data:**

```bash
node scripts/parse.js
# or
npm run build
```

This reads from `the discord/` and writes all JSON data files to `docs/data/`.

**Serve the app:**

```bash
npx serve docs
# or any static file server pointed at docs/
```

Then open `http://localhost:3000`.

## Adding New Exports

1. Drop new HTML export files into `the discord/2019-2025/` or `the discord/2025-2026/` (or a new dated folder)
2. Update the `CHANNELS` source list in `scripts/parse.js` if new channels or filenames appear
3. Re-run `npm run build`
4. Commit and push вЂ” GitHub Pages redeploys automatically

## Deploying to GitHub Pages

The `docs/` folder is the deploy root. In your repo settings:

- **Source**: `Deploy from a branch`
- **Branch**: `main` / `docs` folder

Push to `main` and GitHub Pages serves `docs/` automatically. The `.nojekyll` file is already in place to prevent Jekyll interference.
